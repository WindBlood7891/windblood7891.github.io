Clear UI
一个 可读性强 兼容性强 修改性强 的 SCOS
向下滚动或滑动

可读性强

相比其它作品使用的各种 “自研高级动画引擎” ，Clear UI NEXT 仅涉及了最基础的可打断并行非线性动画，没有复杂公式，让用户轻松看懂代码意思。

兼容性强

作品本身没有使用任何扩展，即使在原版 Scratch 3 也可正常运行。

修改性强

模块化设计，APP 图标和 APP 内容、时间、Clearly 壁纸等内容分开管理，改一处无需处处改。
